# Quickstart (Python)

## Prerequisites

- Python 3.10+

## Install

```bash
python -m pip install -r requirements.txt
```

## Run

```bash
python run.py --mode desktop
```

Then open:

- http://localhost:5000

## Alternate Commands

```bash
python run.py --mode web --host 0.0.0.0 --port 5000
```

Desktop launcher:

```powershell
python waverider_sdr.py
```

Web launcher:

```bash
python waverider_web.py --open-browser
```
